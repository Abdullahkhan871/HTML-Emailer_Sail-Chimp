## ✨ Features

- **Bulletproof Layout:** Built with nested tables for consistent rendering.
- **Fully Responsive:** Uses fluid grids and media queries for mobile optimization.
- **Typography:** Integrated Google Fonts (`Roboto Condensed`) with web-safe fallbacks.
- **Dark Mode Support:** Clean background transitions and high-contrast text.
- **Modular Design:** Easy-to-edit sections for "Email Design," "Email Marketing," and "Email Development."

## 🛠️ Technical Stack

- **HTML:** XHTML 1.0 Transitional
- **CSS:** Inline CSS (for email client compatibility)
- **Fonts:** Roboto Condensed & Sans-Serif fallbacks
- **Layout:** 600px centered container (standard for email)

## 🚀 How to Use

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/sailchimp-email-template.git
    ```
2.  **Open the file:** Open `index.html` in any browser to preview.
3.  **Deployment:** Copy the code into your email service provider (ESP) like Mailchimp, HubSpot, or SendGrid.

## 📂 Project Structure

- `index.html` - The main email template code.
- `img/` - Contains all visual assets used in the email.
- `screen_recording...` - Video showcase of responsiveness.
